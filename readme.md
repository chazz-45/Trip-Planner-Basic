🌍 AI Trip Planner using CAMEL AI

An intelligent AI-powered Trip Planner that generates personalized travel itineraries based on user preferences. The application uses CAMEL AI to simulate collaborative AI agents that work together to create detailed travel plans.

✨ Features

- 🗺️ Personalized travel itinerary generation
- 🤖 Multi-agent collaboration using CAMEL AI
- 📅 Day-wise travel schedule
- 💰 Budget-conscious trip planning
- 🍽️ Food and attraction recommendations
- 🚗 Transportation suggestions
- ⚡ Fast and easy-to-use interface

🛠️ Tech Stack

- Python
- CAMEL AI
- Streamlit (or your frontend)
- OpenAI API / LLM
- Git
- GitHub

🚀 Installation

git clone https://github.com/chazz-45/Trip-Planner-Basic.git
cd Trip-Planner-Basic
pip install -r requirements.txt

Run the project:

streamlit run app.py

(or use the correct command for your project)
